# TP Capstone – Système de Réservation de Salles avec Hibernate

##  Description

Ce projet constitue le TP Capstone final portant sur la conception et l’optimisation d’une application de **réservation de salles** en utilisant **Hibernate** et **JPA**.

L’objectif principal est d’intégrer l’ensemble des concepts avancés étudiés, notamment la gestion des relations entre entités, l’optimisation des performances et la gestion de la concurrence.

---

##  Fonctionnalités principales

- ✅ Modèle de données complet avec relations entre entités
- ✅ Gestion de la concurrence avec *Optimistic Locking*
- ✅ Activation du cache de second niveau (EhCache)
- ✅ Jeu de données réaliste pour les tests
- ✅ Scénarios de test couvrant les fonctionnalités principales
- ✅ Script de migration pour mise à jour de la base de données
- ✅ Rapport d’analyse des performances

---


##  Technologies utilisées

- Java
- Hibernate
- JPA
- EhCache
- MySQL 
- Maven
